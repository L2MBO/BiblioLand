# Политика безопасности

Проект **BiblioLand** — это **локальное десктопное приложение**, не имеющее веб-интерфейса, API или сетевого сервера.  
Все данные хранятся **локально на устройстве пользователя**.

## Поддерживаемые версии

Только **последняя патч-версия** ветки `1.0.x` получает исправления безопасности.  
Пользователям рекомендуется всегда использовать **самую свежую версию**.

| Версия  | Поддержка безопасности |
| ------- | ---------------------- |
| 1.0.2   | ✅ Да                 |
| < 1.0.2 | ❌ Нет                |

## Поверхностные риски
- При компрометации файла базы данных (`.mdf`) возможен доступ к email и аватарам (в Base64).
- Пароли хешируются через **BCrypt** и не могут быть восстановлены.

## Как сообщить об уязвимости
Если вы обнаружили потенциальную уязвимость:
1. Откройте [Issue](https://github.com/L2MBO/BiblioLand/issues).
2. Укажите:
   - Версию приложения,
   - Описание уязвимости,
   - Шаги для воспроизведения.

> ⚠️ Поскольку приложение не имеет сетевых компонентов, большинство классических уязвимостей **не применимы**.

## Поддержка
Проект поддерживается автором в рамках учебной деятельности. Критические проблемы рассматриваются в приоритетном порядке.
